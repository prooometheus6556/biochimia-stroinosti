# ELKA LOUNGE OPERATING PLATFORM

## Описание проекта
Единая платформа управления для заведения **«Ёлка-лаунж на крыше»** (одна локация).
Включает в себя клиентское PWA для бронирования столов и Admin CRM для хостес.

## Технический стек
* **Фреймворк:** Next.js 14 (App Router)
* **Язык:** TypeScript
* **Стилизация:** Tailwind CSS
* **База данных и Auth:** Supabase
* **UI-компоненты:** Radix UI / shadcn/ui

## Дизайн-система "The Luminous Sanctuary"
* **Тема:** Светлая тема (ранее была Dark Theme)
* **Акцентный цвет:** Неоновый лаймовый (#9ffb00)
* **Шрифт:** Manrope
* **Стиль:** Liquid Glass, Bento Grid, Editorial Asymmetry
* **Особенности:**
  - Glassmorphism навигация
  - Неоновые glow-эффекты
  - Бело-лаймовые градиенты
  - Floating bottom navigation bar
  - Noise texture overlay
  - Без 1px borders (только tonal layering)

## Ключевые бизнес-правила
1. **Single Location:** Приложение работает исключительно для «Ёлки на крыше». Выбор заведения отсутствует.
2. **18+ Gate:** Обязательная проверка возраста (18+) при первом входе в приложение.
3. **Правило Омара:** Гостей нельзя ограничивать жестким временем пребывания. Бронирование опирается только на *планируемую* длительность визита.
4. **Авторизация:** Происходит по номеру телефона (OTP), без сложных форм регистрации.
